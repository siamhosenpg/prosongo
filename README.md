## Description

Prosongo is a fully-featured, professional-grade social media application built using **Next.js** for the frontend and **Node.js + Express.js** for the backend, with **MongoDB** for database management. Inspired by **Facebook** and **Instagram**, Prosongo provides a seamless, interactive, and modern social media experience.

It demonstrates a professional full-stack development workflow, combining modern frontend technologies with a robust backend architecture, making it a perfect reference project for aspiring developers or a foundation for production-ready social platforms.

---

## Key Features

- **User Authentication:** Secure signup and login system with JWT-based authentication.
- **Post Management:** Users can create, edit, and delete posts.
- **Interactions:** Multiple reactions (like, love, etc.), comments, comment replies, and sharing functionality.
- **Responsive Design:** Modern UI inspired by Facebook and Instagram for smooth experience across devices.
- **Backend Powered:** All data is securely stored in MongoDB, ensuring scalability and reliability.
- **Optimized Performance:** Implemented best practices for speed and responsiveness.

---

## Technologies Used

- **Frontend:** Next.js, React, tanstack/react-query, axios, framer-motion, wavesurfer
- **Backend:** Node.js, Express.js, cloudinary, bcryptjs, cors, mongoose, multer, sharp
- **Database:** MongoDB
- **Authentication:** JWT (JSON Web Tokens)
- **Styling:** Tailwind CSS / Custom CSS
